# Prompts para ChatGPT Images

Fecha: 2026-03-27

## Uso recomendado

1. Generar primero `concept`.
2. Elegir una sola version por personaje.
3. Recién después pedir variantes de `idle`, `walk`, `attack`, `hurt`, `defeated`.

## Reglas globales para todos

Copiar siempre estas restricciones si ChatGPT empieza a derivar:

```text
Transparent background. Full body. Feet fully visible. Centered character. Same camera angle. No text. No watermark. No anime. No childish cartoon. No fantasy. No sci-fi. Strong readable silhouette for a 2D beat'em up game. Stylized urban caricature. Flat painted game art. Same visual family as the train interior and pickup items. Clean shape language. Simple large shadows. Minimal texture. Simplified clothing folds. Funny but believable. Readable at small size.
```

## Template de variantes

Cuando ya tengas un concept aprobado, usar este prompt base:

```text
Keep the exact same character design, face, clothes, proportions, palette and accessories. Create a full body sprite-style image for a 2D beat'em up game. Pose: <POSE>. Stylized urban caricature, flat painted game art, same visual family as the train interior and pickup items, slightly cartoonish but grounded, expressive proportions, clean shape language, simple large shadows, minimal texture, simplified clothing folds, warm commuter train lighting, transparent background, centered character, feet fully visible, no text, no watermark.
```

Poses:

- `neutral idle pose`
- `idle with slight weight shift`
- `walking pose, left leg forward`
- `walking pose, right leg forward`
- `compact attack pose`
- `hurt recoil pose`
- `defeated pose`
